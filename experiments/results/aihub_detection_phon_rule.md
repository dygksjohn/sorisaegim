# phon_rule(음운규칙) 감지율 + 오탐 대조 — 작업 3b

> 감지 = 엔진이 오류 1개라도 플래그(규칙 위반은 정답발음과의 괴리로 나타남). 정발음 대조군의 플래그율이 오탐(false positive) 기준선.

## 핵심

- **phon_rule 감지율: 94/120 = 78%**
- **정발음 오탐률: 63/120 = 52%** (대조군)
- **순감지 신호(감지−오탐): +26%p**
