# 소리새김 — 후속 계획: ML 고도화 (v1)

> 8주 본 프로젝트 완료 후의 확장 로드맵. 목표는 두 가지다.
> **① 제품 지표**: ~~받침 오류 감지율 35%를 끌어올린다~~ → **[+2주차 실측으로 재정의됨]**
> Whisper 프론트엔드 교체로 **정발음 오탐률 52%를 낮춘다** (아래 📌 참조).
> **② 경력 지표**: fine-tuning(post-training)과 데이터 정제를 "근거 있는 키워드"로 만든다.
> 주당 10~15시간 가정. 본 계획도 v1과 같은 원칙 — 밀리면 '선택'을 버리고 필수만 지킨다.
>
> **📌 표적 재정의 (2026-07-05, +2주차 실측 근거)** — [실측 종합](../experiments/results/aihub_detection_findings.md)
> AI Hub 실발화 측정 결과 전제가 바뀌었다: ① 받침 감지율은 35%가 아니라 **실전 73%**
> (합성 35%는 TTS 한계였음). ② 진짜 병목은 **정발음 오탐률 52%**(정확히 발음한 발화의 절반에
> 가짜 오류)이고, 그 **83%가 Whisper 오인식**이 원인. ③ 따라서 미세조정 표적은 "받침 recall 개선"이
> 아니라 **"Whisper→wav2vec2 발음형 인식기 교체로 오탐 제거 + 표기수렴 미감지 해소"**.
> 정답 라벨은 AI Hub phones 2만 건. before 지표 = 오탐률 52%. 아래 +3주차 이후는 이 표적으로 읽을 것.

## +3~4주차 — wav2vec2 미세조정 (≈24h)

핵심 주. **베이스**: `kresnik/wav2vec2-large-xlsr-korean` (한국어 음향 적응 완료 — 2주차 검증 자산).
**과업**: CTC 헤드를 발음형 자모 vocab으로 교체 후 미세조정. **환경**: Colab GPU (T4/L4), fp16, gradient accumulation.

### 작업
- [ ] **기준선 고정** (3h): 미세조정 전 모델로 테스트셋 평가 — 발음형 CER, 받침 감지율(합성 v2 held-out), 1주차 골드셋 20건. **이 숫자가 모든 비교의 before다.** (`ml/eval_phoneme.py`로 스크립트화 — 이후 반복 사용)
- [ ] **학습 코드** (6h): HF `Trainer` + `Wav2Vec2ForCTC`.
  - vocab: 자모 단위 (초/중/종성 토큰 — engine/hangul의 분해 로직 재사용), feature encoder(CNN) 동결.
  - 1차 하이퍼파라미터: lr 1e-4(헤드)/3e-5(트랜스포머), 5~10 epoch, early stopping(val CER).
- [ ] **1차 학습 + 오류 분석** (5h): val CER 곡선, 혼동 행렬(자모 단위) — 어떤 받침이 여전히 안 잡히는지 2주차 표와 대조.
- [ ] **2차 학습 (개선 1~2가지만)** (5h): 오류 분석 근거로 선택 — 예: 합성 v2 비중 조정, 받침 포함 샘플 오버샘플링, SpecAugment 강화. **한 번에 하나씩 바꾸고 기록** (ablation 표가 곧 이력서 근거).
- [ ] **최종 평가 + 모델 확정** (5h): 테스트셋·골드셋에서 before/after 표 작성. 모델 카드(학습 데이터 구성·지표·한계) 작성 후 저장.

### 완료 기준
- before/after 표가 있다: 발음형 CER, **받침 감지율 35% → X%** (합성 held-out), 골드셋 20건 정성 비교.
- 실패 시에도 완료로 친다 — 단, "왜 안 됐는지"의 오류 분석이 있어야 한다 (과제 C처럼 부정적 결과도 기록).

### 결정 포인트
| 결과 | 결정 |
|---|---|
| 받침 감지율 유의미 개선 (35% → 50%+) | +5주차 통합 진행 |
| CER은 개선됐지만 받침 감지율 정체 | 전역 성능이 아니라 조준 검사에 집중 — +5주차를 "조준 구간 한정 판정"으로 설계 (전역 오인식 노이즈 회피, 2주차 결론 재활용) |
| 학습 자체가 불안정 (CER 발산) | lr 축소 + 헤드만 학습으로 후퇴, 그래도 안 되면 XLS-R 300m에서 재출발 |
