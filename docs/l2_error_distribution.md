# L2 한국어 발음 오류 분포 — 합성 v2 주입 가중치 근거 (+1주차)

> 합성 오발음 데이터 v1의 혼동 쌍은 "그럴듯한" 목록이었다 ([ml_report](ml_report.md) 한계 항목).
> v2는 문헌 근거로 가중치를 정한다. 타깃은 **통합 분포**(특정 모어권 아님 — 주요 모어권 공통
> 오류 중심 + L1 특화 오류 소량), 모어권별 차이는 아래 표로 병기해 이후 모어권별 모드의 재료로 남긴다.

## 1. 근거 문헌

핵심 출처: **ICPhS 2023, "Comparison of L2 Korean pronunciation error patterns from five L1
backgrounds by using automatic phonetic transcription"** ([arXiv:2306.10821](https://arxiv.org/abs/2306.10821),
[논문 PDF](https://www.internationalphoneticassociation.org/icphs-proceedings/ICPhS2023/full_papers/879.pdf))
— 중국어·베트남어·일본어·태국어·영어 5개 모어권, 총 1,500시간 이상의 L2 한국어 발화를
자동 음성전사로 분석한 대규모 연구.

**5개 모어권 공통 오류 패턴 (빈도 상위)**

| 순위 | 패턴 | 예 |
|---|---|---|
| 1 | 격음·경음 → 평음 대체 | ㅋ→ㄱ, ㅌ→ㄷ, ㅍ→ㅂ, ㅊ→ㅈ, ㄲ→ㄱ, ㄸ→ㄷ, ㅃ→ㅂ |
| 2 | 받침(coda) 삭제 | ㄹ 탈락, ㄷ-계열 대표음 탈락 |
| 3 | 이중모음 → 단모음 | ㅟ→ㅣ, ㅞ→ㅔ 등 |

**모어권별 특화 오류 (통합 분포에 소량 반영, 모어권별 모드의 근거)**

| 모어권 | 특화 패턴 | 논문 수치 |
|---|---|---|
| 일본어권 | 받침 ㅇ→ㄴ (/ŋ/→/n/) | 31.2% (전체 평균 10.1%) |
| 일본어권 | ㅜ→ㅓ 모음 대체 | 24.7% (평균 7.1%) |
| 베트남어권 | ㄹ→ㄴ (/l/→/n/) | 23.2% (평균 7.5%) |
| 태국어권 | ㅈ→ㅅ (/tɕ/→/s/) | 22.1% (평균 7.3%) |
| 영어권 | 유의미한 특화 패턴 없음 (공통 패턴 위주) | — |

보조 출처: 국내 음운론적 오류 분석 문헌 일반 —
[외국인 학습자의 한국어 발음 오류에 대한 음운론적 분석 (계명대)](https://newcms.kmu.ac.kr/bbs/actakoreana/424/3370/download.do),
[한국어 학습자의 오류 유형 조사 연구 (국립국어원 한국어교원)](https://kcenter.korean.go.kr/kcenter/attach/download.do?attachSeq=122).
공통 결론: 오류는 모어-목표어 음운 체계 차이에서 나오며, 자음 삼중 대립(평/격/경)과
음절말(받침) 위치가 최대 난점.

## 2. v2 주입 가중치 (synth_data.py에 미러링)

**성분 비중**: 초성 30 / 중성 25 / **받침 45** (v1: 35/35/30).
받침을 올린 이유 — ① 문헌상 받침 삭제가 공통 오류 2위, ② 우리 시스템의 받침 감지율이
35%로 최약 (ml_report 과제 D) → 미세조정 학습·평가 모두 받침 표본이 더 필요하다.

**초성 혼동 쌍** (가중치 — 높을수록 자주 주입):

| from → to | 가중치 | 근거 |
|---|---|---|
| ㅋ→ㄱ, ㅌ→ㄷ, ㅍ→ㅂ, ㅊ→ㅈ | 3 | 격음→평음 (공통 1위) |
| ㄲ→ㄱ, ㄸ→ㄷ, ㅃ→ㅂ, ㅆ→ㅅ, ㅉ→ㅈ | 3 | 경음→평음 (공통 1위) |
| ㄹ→ㄴ | 2 | 베트남어권 특화 (23.2%) |
| ㄱ→ㅋ, ㄷ→ㅌ, ㅂ→ㅍ, ㅈ→ㅊ, ㅅ→ㅆ | 1 | 역방향(과교정·일본어권 경음화 경향) |
| ㄴ→ㄹ, ㅈ→ㅅ | 1 | 소수 패턴 (태국어권 등) |

**중성 혼동 쌍**:

| from → to | 가중치 | 근거 |
|---|---|---|
| ㅓ→ㅗ, ㅗ→ㅓ | 3 | 중국어·일본어권 고전적 혼동 |
| ㅜ→ㅓ | 2 | 일본어권 특화 (24.7%) |
| ㅡ→ㅜ, ㅜ→ㅡ | 2 | 영어권 등 (ㅡ 부재 언어), 과제 D 미감지 쌍 |
| ㅟ→ㅣ | 2 | 이중모음 단모음화 (공통 3위) |
| ㅕ→ㅛ, ㅛ→ㅕ | 1 | 소수 |

**받침 혼동** (삭제 중심):

| from → to | 가중치 | 근거 |
|---|---|---|
| (모든 받침)→삭제 | 5 | 공통 2위 — 특히 ㄹ·ㄷ계열 |
| ㅇ→ㄴ | 3 | 일본어권 특화 (31.2%) |
| ㄴ→ㅇ | 2 | 비음 혼동 (과제 D 미감지 쌍) |
| ㄴ→ㅁ, ㅁ→ㄴ, ㅁ→ㅇ | 1 | 비음 계열 |
| ㄱ↔ㄷ↔ㅂ (대표음 간) | 1 | 소수 |

## 3. v1 대비 제외·기각 규칙 (데이터 정제 결정)

1. **ㅐ↔ㅔ 쌍 제외** — 현대 서울말에서 두 모음은 병합되어 TTS가 동일하게 발음한다.
   주입해도 오디오가 정상본과 같아 **라벨만 틀린 데이터**가 된다. v1에 있었으나 v2에서 제거.
2. **무효 변형 기각** — 치환 후 표기의 g2pk 발음형이 원문과 같으면(예: "같이"→"가치",
   구개음화로 동음) TTS 출력이 동일해 라벨 오류가 된다. **v1에서 실제로 5건 검출**
   (data_quality_report.csv의 synth duplicate_exact) → v2 생성기는 `g2pk(변형)==g2pk(원문)`이면
   재시도한다.
3. **없는 받침에 받침 추가는 계속 배제** — 삽입 오류는 L2 문헌에서 소수이고(일본어권 모음 삽입은
   음절 삽입이라 자모 치환으로 표현 불가), 라벨 해석이 복잡해진다. v3 과제로 남긴다.
4. **생성 후 PCM 해시 검사가 최종 방어선** — g2pk 텍스트 검사로 못 잡는 무효 변형이 있다.
   TTS가 자체 음운 규칙을 적용하는 경우로, v2 실측 1건: "활짝"→"활작"(ㅉ→ㅈ)을 TTS가
   경음화로 둘 다 [활짝]으로 발음 → 오디오 동일. `data_quality.py`의 정확 중복 검사가
   검출해 fail 처리하며, 데이터셋 구축은 pass만 사용한다 (텍스트 필터 → 음향 필터 2단 구조).

## 4. 한계 (정직하게)

- 가중치 3/2/1은 논문의 정확한 빈도 재현이 아니라 **순위·배수 관계를 보존한 근사**다.
  논문은 음소(전사) 단위, 우리는 자모(표기) 단위라 1:1 대응이 안 되는 패턴이 있다.
- 통합 분포는 5개 모어권 코퍼스 규모에 비례하지 않는다 (영어권 24h vs 중국어권 487h).
  "주요 모어권 공통 + 특화 소량"이라는 설계 선택이며, 모어권별 모드가 후속 과제.
- 이중모음 오류 중 ㅘ→ㅏ 같은 활음 탈락은 우리 문장 세트에 해당 음절이 적어 반영을 보류했다.
