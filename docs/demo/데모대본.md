# 데모 영상 촬영 대본 (2분 컷)

> 녹화: 게임바(Win+G) — 마이크 충돌은 해결됨(원시 캡처). 브라우저는 Ctrl+F5 후 시작.
> 서버: `.venv\Scripts\uvicorn.exe app.main:app --port 8000` (미리 한 문장 채점해 모델 예열)
> 자막을 넣을 거라면 육성 해설은 생략해도 됨 — 발화는 연습 문장 읽기만으로 충분.

| 구간 | 화면 | 행동 | 보여줄 것 |
|---|---|---|---|
| 0:00–0:10 | 연습 화면 | 문장 카드 잠시 정지 | 제목·문장·발음형·유형 배지 (자막: "어디가 틀렸는지 짚어주는 발음 코치") |
| 0:10–0:25 | 연습 화면 | [🔊 듣기] 클릭 | TTS 표준 발음 재생 |
| 0:25–0:45 | 연습→결과 | [🎙 녹음] → **정상 발음** → 정지 | 채점 스피너 → **100점** (초록) |
| 0:45–1:15 | 연습→결과 | [다시 도전] → 같은 문장을 **일부러 틀리게** (예: "밥을"을 [받을]로) | 점수 하락 + **틀린 음절 빨간 하이라이트** → 마우스 올려 **툴팁** ("기대: ㅂ / 발음됨: ㄷ") + "이렇게 들렸어요" 비교 |
| 1:15–1:35 | 학습 기록 | 상단 [📊 학습 기록] 클릭, 천천히 스크롤 | 점수 추이 → 유형별 평균 → **자주 틀리는 자모 TOP 5** |
| 1:35–1:55 | 연습 화면 | 돌아와서 [🎯 추천 문장] 클릭 | 추천 이유 표시 ("오류 확률 %·취약 자모 포함...") → 그 문장 바로 녹음 1회 |
| 1:55–2:00 | 결과 화면 | 점수 확인 후 정지 | (자막: "규칙 기반 대비 추천 정확도 0.4→0.6 · GitHub 링크") |

**팁**
- 틀리게 읽을 문장은 1주차에 검증된 것이 안전: "밥을 먹었다"→[받을], "같이 갑시다"→[갇이]
- 4주차 원본 영상(`mvp_demo_2026-07-04.mp4`)에서 쓸만한 컷이 있으면 재활용 가능하지만,
  추천 버튼·원문 표시가 없던 버전이라 새로 찍는 걸 권장
- 편집 없이 한 번에 가려면 위 순서로 리허설 1회 후 본촬영
